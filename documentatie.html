Kid Web Monitor

Sistem de monitorizare în timp real a copiilor în spațiul urban

Rezumat

Proiectul Kid Web Monitor propune un sistem web modern care permite monitorizarea în timp real a copiilor dintr-o familie. Funcționalitățile principale includ urmărirea locației copilului pe hartă, detectarea interacțiunilor apropiate între copii, alerte de distanțare față de puncte de interes și simulări dinamice. Interfața este responsivă, iar aplicația este construită folosind HTML, CSS, JavaScript, AJAX și un backend PHP REST fără framework-uri. Datele sunt stocate într-o bază de date MySQL.

Structura aplicației

Proiectul conține următoarele pagini: login, înregistrare, dashboard (monitorizare), și simulare. Sistemul permite autentificarea părinților, urmărirea în timp real a poziției copiilor și alertarea automată în caz de deviere față de punctele de interes sau interacțiuni riscante între copii. Toate componentele sunt create fără framework-uri, cu o abordare modulară în foldere (e.g., `api/children`, `api/users`).
